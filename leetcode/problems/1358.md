# [1358. Number of Substrings Containing All Three Characters](https://leetcode.com/problems/number-of-substrings-containing-all-three-characters)

## Approach (Using Map)
I needed to find all the substring containing 'a', 'b', and 'c'. 
If I find the shortest substring containing atleast 1 'a', 'b', and 'c' all the string concatinated with that will definately satisfy the condition so I' checking it with map and once satisfied, I'm taking all the substring afterwards, once done, I'll pop one character from front.

## Code
```python
from collections import defaultdict
class Solution:
    def numberOfSubstrings(self, s: str) -> int:
        total = 0
        n = len(s)
        d = defaultdict(int)
        last = 0
        def check():
            if d['a'] < 1 or d['b'] < 1 or d['c'] < 1:
                return False
            return True
        for i in range(n):
            d[s[i]] += 1
            while check():
                total += (n-i)
                d[s[last]] -= 1
                last += 1
        return total
``` 

## Approach (Using constant Space)
Instead of keeping track and adding all the extra string toward end, I took three variables and track 'a', 'b' and 'c'. Once find I took the left most point and similarly all the string in left side will be the solution.
     
## Code
```python
class Solution:
    def numberOfSubstrings(self, s: str) -> int:
        res = 0
        a = b = c = -1
        for i, ch in enumerate(s):
            if ch == 'a':
                a = i
                m = min(b, c)
            elif ch == 'b':
                b = i
                m = min(a, c)
            else:
                c = i
                m = min(a,b)
            res += 1 + m
        return res
        
```
