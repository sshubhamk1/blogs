# [208. Implement Trie (Prefix Tree)](https://leetcode.com/problems/implement-trie-prefix-tree)

## Problem Statement

A **Trie** (pronounced as "try") or **Prefix Tree** is a specialized tree data structure used to efficiently store and retrieve keys in a dataset of strings. Unlike a binary search tree, nodes in a Trie do not store the key itself; instead, a node's position in the tree defines the key it is associated with.

Tries are widely used in various real-world applications, such as:
* **Autocomplete/Type-ahead suggestions:** Predicting the rest of a word as you type.
* **Spell Checkers:** Verifying if a word exists in a dictionary.
* **IP Routing:** Matching longest prefixes in networking tables.

Implement the `Trie` class:

* `Trie()` Initializes the trie object.
* `void insert(String word)` Inserts the string `word` into the trie.
* `boolean search(String word)` Returns `true` if the string `word` is in the trie (i.e., it was previously inserted), and `false` otherwise.
* `boolean startsWith(String prefix)` Returns `true` if there is a previously inserted string that has the prefix `prefix`, and `false` otherwise.

---

## Complexity Analysis

Let $L$ be the length of the word being inserted, searched, or prefix-checked.

### Time Complexity
| Operation | Complexity | Description |
| :--- | :--- | :--- |
| `insert` | $O(L)$ | We iterate through every character in the word once. |
| `search` | $O(L)$ | We traverse the tree down to the length of the word. |
| `startsWith` | $O(L)$ | We traverse the tree down to the length of the prefix. |

### Space Complexity
* **Space Complexity:** $O(N \times L)$, where $N$ is the total number of words inserted and $L$ is the average length of the words. In the worst case, every character of every word requires a new node in the Trie.

---

## Approach

The core idea behind a Trie is to represent each character of a string as a node in a tree. To implement this, we use a helper `Node` class. Each node contains a dictionary (or hash map) where:
1. The **keys** are the characters (e.g., `'a'`, `'b'`, `'c'`).
2. The **values** are pointers to the next `Node`.
3. A special marker (in this implementation, `None`) is used within the dictionary to indicate that a valid word ends at that specific node.

### Logic Breakdown:
1. **Insertion:** We traverse the tree character by character. If a character does not exist as a child of the current node, we create a new node. Once we reach the end of the word, we mark the node by adding `None` as a key in its dictionary.
2. **Searching:** We traverse the tree based on the characters of the input word. If at any point a character is not found in the current node's children, the word does not exist. If we reach the end of the word, we check if the `None` key exists in the current node's dictionary to ensure it's a complete word and not just a prefix.
3. **Prefix Search:** This is similar to searching, but we don't care if the `None` key exists. We only care if the path for the prefix exists in the tree.

## My Solution

```python
class Node:
    """Represents a single node in the Trie."""
    def __init__(self):
        # children maps a character to the next Node
        # A special key None is used to mark the end of a valid word
        self.children = {}

class Trie:
    def __init__(self):
        """
        Initializes the trie object.
        """
        self.root = Node()

    def insert(self, word: str) -> None:
        """
        Inserts the string word into the trie.
        """
        current_node = self.root
        for char in word:
            # If character is not present, create a new node
            if char not in current_node.children:
                current_node.children[char] = Node()
            current_node = current_node.children[char]
        
        # Mark the end of the word using None as a special terminal key
        current_node.children[None] = True

    def search(self, word: str) -> bool:
        """
        Returns true if the string word is in the trie, and false otherwise.
        """
        current_node = self.root
        for char in word:
            # If character is not found, the word doesn't exist
            if char not in current_node.children:
                return False
            current_node = current_node.children[char]
        
        # Return True only if the None key exists (signifying end of a word)
        return current_node.children.get(None, False)

    def startsWith(self, prefix: str) -> bool:
        """
        Returns true if there is a previously inserted string that has the given prefix.
        """
        current_node = self.root
        for char in prefix:
            # If character is not found, no word starts with this prefix
            if char not in current_node.children:
                return False
            current_node = current_node.children[char]
        
        # If we successfully traversed the prefix, return True
        return True
```
