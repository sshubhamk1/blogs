## Reporting a Vulnerability

We take security seriously. If you discover a vulnerability in this repository, please do not open a public issue. Instead, please report it privately by [Email Address/Method].

We will investigate the issue and work on a fix as quickly as possible. Once the issue is resolved, we will notify you and potentially publish a security advisory.

## Scope
This policy applies to the codebase and documentation within this repository.
