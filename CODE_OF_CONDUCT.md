# Contributor Covenant Code of Conduct

## Our Pledge
In the interest of fostering an open and welcoming environment, we as contributors and maintainers pledge to every person using this software to make a significant contribution to the community. To this end, we pledge that all contributors for this project are expected to uphold this Code of Conduct.

## Our Standards
Contributors are expected to follow these Code of Conduct guidelines:
* **Be respectful, kind, and inclusive.**
* **Respect privacy and avoid using personal information without consent.**
* **Avoid harassment and other forms of bullying.**
* **Be professional and focus on technical/contributor discussion.**

## Enforcement
Examples of behavior that are considered violations of this code include:
* Unwelcome sexual attention or advances.
* Insults or derogatory comments about race, religion, culture, identity, or experience.
* Insults or derogatory comments about sexual orientation, gender identity, or gender expression.
* Disrespectful or aggressive behavior.
* Piercing, mocking, or intimidating a contributor.

## Enforcement Responsibilities
Maintainers are responsible for enforcing this Code of Conduct. If you are the first person to observe a violation, please contact the project maintainers and welcome them to do so.

## Scope
This Code of Conduct applies within all project spaces and is a reflection of the community's commitment to maintaining a welcoming and healthy environment.

## Enforcement
Instances of abusive, harassing, or otherwise unacceptable behavior may be reported by any community member to the project maintainers and will be reviewed promptly and reliably where possible.
